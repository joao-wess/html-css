Negrito

Nesse parágrado terá palavras em negrito usando a tag <b> e </b> (NAO SEMANTICA)

Neste parágrafo, terá palavras em negrito usando a tag <strong> e </strong>(COM SEMANTICA)

Itlático

Nesta frase, temos palavras em itálico usando a tag <i> e </i>(NAO SEMANTICA)

Nesta frase, temos palavras em itálico usando a tag <em> e </em> (COM SEMANTICA)

Texto Marcado

Nesta frase, haverá frases marcadas usando a tag mark

Nesta outra frase também haverá palavras marcadas

Texto grande e pequeno

aqui há palavras grandes usando a tag <big>
que já se encontra obsoleta e palavras pequenas usando a tag <small>

Texto deletado

Podemos marcar um texto como excluído, onde indica que ele pode ser lido mas não considerado usando a tag <del>

Texto Inserido

Podemos também usar a tag <ins> para dar ênfase e indicar que foi adicionado depois

Texto sobrescrito

Para mostrar coisas como x20+3 (USA-SE A TAG <sup>)

Texto subescrito
